# Homebrew Entigram

Official Homebrew Tap for **Entigram**, the schema-first semantic governance layer for enterprise agents.

## 📦 Installation

To install Entigram using this tap, run the following commands in your terminal:

```bash
brew tap nyabutid/entigram
brew install etg
```

## 🛠️ Usage

Once installed, you can use the trusted core `etg` CLI globally:

```bash
etg --help
```

The Homebrew formula intentionally installs the core CLI from declared,
checksummed resources only. To use the Streamlit Visual Dashboard, install the
Python package directly:

```bash
pipx install entigram-ai
```
